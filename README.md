📝 App Treino - Personal Digital
Este projeto é uma interface web moderna e responsiva desenvolvida para gestão de treinos, conectando alunos e personal trainers de forma intuitiva. O design foi inspirado em padrões de aplicativos mobile de alta performance, com foco em usabilidade e estética "premium".

🚀 Funcionalidades Atuais
Header Dinâmico: Saudação personalizada ao aluno e exibição da marca/foto do Personal Trainer.

Silhueta Customizável: Suporte para exibição de silhueta (PNG transparente) do treinador com efeito de sobreposição.

Calendário Semanal Automático: Gerado via JavaScript, destacando automaticamente o dia atual da semana.

Grid Responsivo: Layout que se adapta de 2 colunas (Mobile) até 4 colunas (Desktop), otimizando o espaço da tela.

Painel de Teste: Interface temporária para simulação de troca de dados (nomes e fotos) em tempo real.

🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica dos componentes.

CSS3: Estilização avançada, CSS Grid, Flexbox e Gradientes Radiais.

JavaScript (Vanilla): Lógica para manipulação do DOM e geração dinâmica de datas.
